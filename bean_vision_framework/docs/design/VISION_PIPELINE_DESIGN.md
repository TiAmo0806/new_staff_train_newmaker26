# RM物流赛视觉系统设计方案

版本：V1.0  
用途：进入实车联调阶段前的视觉系统设计冻结文档

---

# 1. 系统目标

视觉系统负责：

- 豆子识别
- 数字箱识别
- 豆子与数字绑定
- 生成最终任务结果
- 与电控完成通信闭环

视觉系统**不负责**：

- 云台角度计算
- 云台姿态解算
- 电机控制

云台运动由电控完成，视觉只请求固定动作并等待反馈。

---

# 2. 总体比赛流程

整体状态：

```text
等待启动

    ↓

等待到达豆子区域

    ↓

豆子识别

    ↓

缓存豆子结果

    ↓

等待到达数字区域

    ↓

进入数字识别流程

    ↓

多视角数字扫描

    ↓

数字结果融合

    ↓

生成最终任务

    ↓

发送结果
```

---

# 3. 豆子识别流程

## 3.1 特点

豆子区域：

- 三个固定ROI
- 一次图像完成识别
- 不需要云台运动


流程：

```text
BEAN_VIEW

    ↓

YOLO检测

    ↓

ROI解析

    ↓

P1/P2/P3绑定

    ↓

结果缓存
```

---

## 3.2 输出结果

示例：

```text
P1:
soybean

P2:
mung_bean

P3:
white_kidney_bean
```

# 4. 数字识别设计

## 4.1 空间布局

数字箱实际布局：

```text
            后方


        箱3   箱2   箱1


    箱5              箱4



            车辆
```

特点：

- 共5个数字箱
- 箱子数字面方向不同
- 单一视角无法覆盖全部数字
- 同一个箱子可能在多个视角出现


---

# 5. 云台运动逻辑

## 5.1 Pitch轴

作用：

进入数字观察区域。


流程：

```text
豆子识别姿态

        |

        | pitch旋转约120°

        ↓

数字观察姿态
```


特点：

- 电控控制
- 视觉等待完成通知


---

## 5.2 Yaw轴

作用：

改变观察角度，完成数字扫描。


特点：

- 固定三个角度
- 比赛前电控调参
- 视觉不知道具体角度


视觉只认：

```cpp
enum class DigitView
{
    VIEW0,
    VIEW1,
    VIEW2
};
```

---

# 6. 多视角数字识别

三次识别不是重复检测，而是：

> 不同空间视角获取信息，最终融合。


示例：

## VIEW0

```text
箱1 = 1
箱2 = 2
箱3 = 3
```


## VIEW1

```text
箱2 = 2
箱3 = 6
箱5 = 5
```


## VIEW2

```text
箱1 = 1
箱4 = 4
```


最终：

```text
箱1 = 1
箱2 = 2
箱3 = 3
箱4 = 4
箱5 = 5
```

# 7. View设计

禁止：

```text
根据yaw角动态计算ROI
```

原因：

- 无云台姿态反馈
- 无IMU闭环


采用：

```text
固定观察状态 → 固定ROI
```

结构：

```text
View

  ↓

对应ROI

  ↓

检测

  ↓

融合
```

---

# 8. ROI设计

ROI属于观察状态，而不是箱子。


推荐：

```yaml
digit_views:

  VIEW0:
    box1:
    box2:
    box3:


  VIEW1:
    box2:
    box3:
    box5:


  VIEW2:
    box1:
    box4:
    box5:
```

---

# 9. 数字结果缓存

不能直接覆盖识别结果。

采用：

```cpp
struct DigitObservation
{
    int box_id;

    int digit;

    float confidence;

    DigitView view;
};
```

保存：

```text
VIEW0结果

VIEW1结果

VIEW2结果
```

---

# 10. 数字融合策略


目标：

输出：

```text
box1
box2
box3
box4
box5
```

融合原则：

1. 同一个箱子多结果比较
2. 高置信度优先
3. 多视角一致优先
4. 冲突严重时重新识别


例如：

```text
箱2:

2  confidence=0.92

6  confidence=0.55
```

选择：

```text
2
```


如果：

```text
2  confidence=0.91

6  confidence=0.89
```

认为：

```text
冲突
```

需要额外处理。
