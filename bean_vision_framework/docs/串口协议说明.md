# 串口协议说明

当前串口协议文档拆分为三版，分别面向不同读者：

## 1. 核心协议

- [串口协议_核心协议.md](./串口协议_核心协议.md)
  - 以当前代码实现为准
  - 描述帧格式、命令字、编码表、payload 结构
  - 适合需要核对 `Protocol.cpp` / `SerialPort.cpp` 的开发者

## 2. 电控端协议

- [串口协议_电控端.md](./串口协议_电控端.md)
  - 面向 C 板 / 电控开发
  - 强调“应该发什么、会收到什么、收到后怎么判定 ACK”
  - 适合写串口接收解析和状态联调

## 3. 算法端协议

- [串口协议_算法端.md](./串口协议_算法端.md)
  - 面向本仓库主程序和调试工具
  - 强调“什么时候发包、什么时候等 ACK、失败如何处理”
  - 适合视觉/算法侧联调

## 4. 代码对应位置

- 协议定义：[include/communication/Protocol.h](../include/communication/Protocol.h)
- 协议实现：[src/communication/Protocol.cpp](../src/communication/Protocol.cpp)
- 串口收发：[src/communication/SerialPort.cpp](../src/communication/SerialPort.cpp)
- 串口命令转换：[src/command/SerialCommandSource.cpp](../src/command/SerialCommandSource.cpp)
