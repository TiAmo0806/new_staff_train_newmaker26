# 三、协议格式（已验证）

## 3.1 数据帧结构

```
HEAD | CMD | LENGTH | SEQ | PAYLOAD | CRC16_L | CRC16_H
```

字段：

|字段|长度|说明|
|---|---|---|
|HEAD|1B|固定0xA5|
|CMD|1B|命令类型|
|LENGTH|1B|payload长度|
|SEQ|1B|序号|
|PAYLOAD|0~32B|数据|
|CRC16|2B|校验|

总长度：

```
6 + payload长度
```

---

# 四、CRC协议

双方一致：

```
CRC16-MODBUS
```

参数：

|项目|值|
|---|---|
|算法|CRC16-MODBUS|
|初始值|0xFFFF|
|多项式|0xA001|
|输出异或|0x0000|
|字节序|低字节先发送|

校验范围：

```
HEAD

↓

PAYLOAD最后一个字节
```

不包含：

```
CRC自身
```
